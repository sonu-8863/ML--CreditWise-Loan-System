# ML--CreditWise-Loan-System
Developed a machine learning classification model using Gaussian Naive Bayes after performing detailed Exploratory Data Analysis (EDA) on the dataset. Data preprocessing steps included handling missing values using SimpleImputer, encoding categorical variables with LabelEncoder and OneHotEncoder, and feature scaling using StandardScaler.

The dataset was split into training and testing sets using train_test_split, and the model was trained and evaluated using multiple performance metrics. The model achieved an accuracy of 86.5%, with precision of 0.80, recall of 0.73, and F1-score of 0.76, demonstrating strong classification performance.

Additionally, a confusion matrix was used to analyze prediction errors and validate model effectiveness. The project highlights the ability to build end-to-end machine learning pipelines, perform data analysis, and evaluate models effectively.
